# LambdaMotd

A plugin for Spigot servers which allows you to set a custom motd in the server list with advanced features.

## Features

 - MOTD:
   * Basic with variables
   * Random
 - Player cache:
   * Player name in the MOTD
   * Player icon
 - Favicon:
   * Basic with server-icon.png
   * Random
   * Player favicon
 - Playercount:
   * Custom version.
   * Custom playercount
     - Random values
 - Custom playerlist:
 - Commands:
   * reload
